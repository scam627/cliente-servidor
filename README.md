# Installing ZMQ

- **Python**

In anywhere location run:

```bash
pip3 install pyzmq
```

- **Node.js**

In js directory run:

```bash
npm i zeromq@5
```

# Test

- **Nodejs:**

In js directory execute:

```bash
node server.js
```

Open another terminal and run:

```bash
node client.js

```

- **Python:**

In python directory execute:

```bash
python3 server.py
```

Open another terminal and run:

```bash
python client.py

```
